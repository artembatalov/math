# 📐 Конспекты: Математический анализ и СРВМ

TeX-конспекты, структурированные **по разделам экзаменационных списков вопросов**.
Один раздел программы — один файл. Для каждого раздела доступны исходник `.tex` и
скомпилированный `PDF` (кнопки в таблицах ниже).

> Источник — рукописные лекционные конспекты (`analysis/*.pdf`, `advanced/*.pdf`).
> При переносе в TeX сохранён стиль изложения (Определение / Теорема / Доказательство),
> доведены до конца доказательства и заполнены пропуски (`todo`) оригинала.

## ⚙️ Как скомпилировать

Файлы рассчитаны на **XeLaTeX / LuaLaTeX** (кириллица через `fontspec` + шрифт
*CM Unicode*, локализация через `polyglossia`). Преамбула вшита в каждый файл, поэтому
любой `.tex` компилируется самостоятельно.

```bash
# вариант 1: tectonic (ставится одним бинарником, сам докачивает пакеты)
tectonic analysis/tex/01-pervoobraznaya-neopr-integral.tex

# вариант 2: классический TeX Live
xelatex analysis/tex/01-pervoobraznaya-neopr-integral.tex
```

На **Overleaf**: загрузить `.tex`, в *Menu → Compiler* выбрать **XeLaTeX**.

> 💡 Кнопки `TeX` / `PDF` ведут на файлы в репозитории. После публикации на GitHub
> они работают как ссылки на скачивание (кнопка **Download raw file** на странице файла).
